# Expense Tracker - Трекер расходов

## Автор
**Иван Петров** Негребова Кристина

## Описание программы
Expense Tracker — это десктопное приложение для отслеживания личных финансов. Программа позволяет добавлять расходы, фильтровать их по категориям и датам, а также подсчитывать общую сумму расходов за выбранный период. Все данные сохраняются в JSON-файл и автоматически загружаются при запуске.

## Возможности
- Добавление расходов с указанием суммы, категории и даты
- Валидация ввода (сумма > 0, корректный формат даты)
- Отображение всех расходов в таблице
- Фильтрация по категории и диапазону дат
- Подсчёт суммы расходов за отфильтрованный период
- Сохранение и загрузка данных в JSON
- Интуитивно понятный графический интерфейс
